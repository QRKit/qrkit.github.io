QRKit X Privacy Policy

Last updated: January 31, 2026

Important: QRKit X (hereinafter referred to as "this app") highly values user privacy protection. This privacy policy aims to explain how we collect, use, and protect your personal information. Please read this policy carefully before using this app.

1. Introduction

QRKit X (hereinafter referred to as "this app") highly values user privacy protection. This privacy policy aims to explain how we collect, use, and protect your personal information. Please read this policy carefully before using this app.

2. Information Collection

Types of information collected by this app:

  • Camera Permission: Used for scanning QR codes. We do not save any images captured by the camera. Camera data is only used in real-time during scanning.
  • Storage Permission: Used for saving generated QR code images to gallery. Only used when user actively saves.
  • Local Data: Scan and generation history records, app settings (such as theme settings). All data is stored locally on device only.

3. Data Usage

Collected information is used for:

  • Providing QR code scanning and generation services
  • Saving history records and settings
  • Improving app experience

We promise:

  • Not to upload data to servers
  • Not to share personal data with third parties
  • Not to collect device identifiers for tracking

4. Data Storage and Security

Data Storage:

  • All data is stored locally on device only
  • Uninstalling the app will delete all local data

Data Security:

  • This app does not perform network transmission
  • Data security depends on device security

5. User Rights

You have the following rights:

  • Delete history records within the app at any time
  • Manage app permissions in system settings
  • Uninstall the app to delete all local data
  • Contact us to learn about data processing details

6. Children's Privacy

This app is suitable for users of all ages. We do not intentionally collect personal information from minors. If you are a minor, please use this app under the guidance of a guardian.

7. Privacy Policy Updates

This privacy policy may be updated from time to time. Updated policies will be published within the app with the update date noted. We recommend that you regularly review this policy to understand how we protect your information.
