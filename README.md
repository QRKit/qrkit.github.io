# 便捷二维码 - 法律文档

本目录包含便捷二维码应用的法律文档，包括隐私政策和用户协议。

## 📁 文件说明

- **index.html** - 首页，提供文档导航
- **privacy-policy.html** - 隐私政策
- **terms-of-service.html** - 用户协议

## 🌐 部署到服务器

你需要将这些 HTML 文件部署到一个可访问的服务器上。

### 部署选项

1. **使用云服务器**（推荐）
   - 阿里云 OSS
   - 腾讯云 COS
   - 华为云 OBS
   - 七牛云

2. **使用虚拟主机**
   - 购买虚拟主机服务
   - 通过 FTP 上传文件

3. **使用其他静态网站托管服务**
   - Vercel
   - Netlify
   - Cloudflare Pages

## 📝 使用说明

### 在华为应用市场填写

部署完成后，在华为应用市场填写：

- **隐私政策链接**：`https://你的域名/privacy-policy.html`
- **用户协议链接**：`https://你的域名/terms-of-service.html`

### 在应用内使用

如果需要在应用内显示这些文档，可以使用 WebView 加载：

```typescript
// 示例代码
const privacyUrl = 'https://你的域名/privacy-policy.html';
const termsUrl = 'https://你的域名/terms-of-service.html';
```

## ✏️ 自定义内容

在发布前，请务必修改以下占位符：

### 在所有 HTML 文件中

1. **开发者信息**
   - `[您的名字或公司名]` → 替换为你的真实姓名或公司名

2. **日期信息**
   - 确认更新日期和生效日期是否正确


## 🔍 本地预览

在部署前，可以在本地预览：

### 方法 1：使用 Python

```bash
cd docs
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

### 方法 2：使用 Node.js

```bash
cd docs
npx http-server
```

### 方法 3：直接打开

```bash
open index.html
```

## 💡 提示

1. **隐私政策和用户协议是必需的**

   - 链接必须可以正常访问
   - 内容必须真实、完整

2. **定期更新**
   - 如果应用功能有重大变更，记得更新这些文档
   - 更新后记得修改文档顶部的更新日期

3. **保持简洁**
   - 文档应该简洁明了，易于用户理解
